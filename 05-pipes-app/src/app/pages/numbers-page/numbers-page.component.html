Pipe number

El pipe number en Angular se utiliza para formatear números como texto, permitiendo definir la cantidad de decimales y el formato de separación de miles. Es útil para mostrar cantidades de manera legible y estandarizada. Se aplica usando el símbolo | seguido de number en el template.

Ejemplo:

{{'{{ totalventa() | number:"1.2-2" }}'}}

Valor original: {{totalventa()}}
{{totalventa() | number: '1.2-2' }}
Pipe currency

El pipe currency en Angular se utiliza para formatear un número como una cantidad monetaria, mostrando el símbolo de la moneda y el formato adecuado según la configuración regional. Se aplica usando el símbolo | seguido de currency y el código de la moneda en el template.

Ejemplo:

{{'{{ totalventa() | currency:"COP" }}'}}

Valor original: {{totalventa()}}
{{totalventa() | currency: 'CAD' }}
Pipe percent

El pipe percent en Angular se utiliza para mostrar un número como porcentaje, multiplicando el valor por 100 y agregando el símbolo %. Se aplica usando el símbolo | seguido de percent en el template.

Ejemplo:

{{'{{ porcentajeVenta() | percent:"1.2-2" }}'}}

Valor original: {{porcentajeVenta()}}
{{porcentajeVenta() | percent:'1.2-2'}}
